Project
-------
Apllication icon is set
Assembly Info
Warnings as Errors = All (no exclusions)

Compiler
--------
- Set warning as errors
- Make sure that Debug info is either set to full for debug builds or to pdb-only for release builds. It should not be set to none.

Code Analysis
-------------
- Get to zero violations


Style Analysis
--------------
- Install StyleCop
- Use all rules
- Get to 100% style conformance

Code Coverage
-------------
- Get to 100% coverage
- Use [ExpectedException(typeof(T))] co catch expected exceptions
- Unit tests that throw an exception won't get 100% coverage (everything after the exception is dead code), so use the Attribute: [ExcludeFromCodeCoverage]
